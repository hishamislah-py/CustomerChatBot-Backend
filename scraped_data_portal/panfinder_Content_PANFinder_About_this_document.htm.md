# About this document

This guide describes the Thredd PANFinder app, which can be used to retrieve the full Primary Account Number (PAN) of a card.

## Target Audience

This guide is aimed at users such as Payment Card Administrators, Customer Service Specialists, and Card Fraud Risk Managers.
