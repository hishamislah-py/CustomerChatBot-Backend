# Regenerate the Wallet

API: `Ws_RegenerateWallet`

This web service regenerates an [MFX card![Closed](../../Skins/Default/Stylesheets/Images/transparent.gif) Payment card which supports payment and settlement transactions in multiple currencies. The MFX card typically has a single PAN with multiple currency wallets linked.](#). For regenerating the wallet, you can select whether to create a new physical linked card, regenerate the image for the virtual card or return the CVV.
