# 21 Creating Reports and Queries

The Analytics menu on the [Apata Portal Dashboard](Apata_dashboard.htm#Apata_Portal_Dashboard) enables you to create custom reports and build queries on you 3D Secure transactions.

Reports can be sent to a list of email addresses or uploaded to an [SFTP![Closed](../../Skins/Default/Stylesheets/Images/transparent.gif) Secure File Transfer Protocol provides a means of transferring files to a secure server.](#) Server.

## 21.1 Viewing and Generating Reports

1. To view a list of available reports, select Analytics > Reports from the left-hand menu.

   [![](../Resources/Images/Apata/reports.png "Reports screen")](../Resources/Images/Apata/reports.png)

   Figure 16: Viewing Reports
2. To view information about the report, click the ![](../Resources/Images/Apata/View_details.png) icon.
3. To change details in the report, click the Edit button. After you have made your changes, click Update Report.
4. To generate a new report, click the Report Builder button.
5. Add a Name and Description to your report, select the type of Query and report Schedule (Daily, Weekly, Monthly).
6. If you want to deliver reports to an email address:

   1. Select the **Enable automatic report delivery** option and enter the user's email address.
   2. To add additional email addresses, click the **+ Email** button.

SFTP configuration is currently not available

[![](../Resources/Images/Apata/report_builder.png "Report Builder")](../Resources/Images/Apata/report_builder.png)

Figure 17: Report Builder

The report is added to the list of available reports. (If you cannot see it in the list, refresh the page.)

## 21.2 Viewing and Building Queries

1. To view a list of available transaction queries, select Analytics > Queries from the left-hand menu.

[![](../Resources/Images/Apata/queries.png "Reports screen")](../Resources/Images/Apata/queries.png)

Figure 18: Viewing Queries

2. To view details of a query, click the ![](../Resources/Images/Apata/View_details.png) icon.
3. To build a new query, click the Query Builder button.
