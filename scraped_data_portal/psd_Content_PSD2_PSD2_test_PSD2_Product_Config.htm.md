# PSD2 Transaction Checker

Find out if the PSD2 rules apply to a transaction and whether the transaction will be approved or declined, using our transaction checker wizard.

## Product and SCA Config Checks

How has SCA been set up for your cards? Tick one of the following:

The Thredd Product is not enabled for PSD2 and SCA

The Wallet Provider (e.g., Apple Pay) manages the PSD2 checks

The transaction was Strong Customer Authenticated (SCA)

None of the above apply

No further PSD2 checks are required. The transaction will be approved (provided that there is no other reason to decline).
