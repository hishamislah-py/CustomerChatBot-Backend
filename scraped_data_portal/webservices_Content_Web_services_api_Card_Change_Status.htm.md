# Card Change Status

API: `Ws_StatusChange`

This web service enables you to change the status of a card, for example to: *Do not honor*, *lost*, *stolen* and *Card destroyed*. For details see [Status Codes](../Reference/Status_Codes.htm).

Changes made to the card's status will also update the status of any linked digital payment tokens if the option *dpan\_over\_fpan = 芒聙聹N芒聙聺* is set for your cards. For details, see the [Tokenisation Service Guide](https://docs.thredd.com/Tokenisation_Guide.htm).
