# About this Guide

This guide describes the card payments ecosystem and how Thredd supports your card program.

## Target Audience

Technical teams responsible for setting up a new card program.

## What's Changed?

If you want to find out what's changed since the previous release, see the [Document History](../Document_History.htm) section.

## How to use this Guide

* If you are new to card payments and want to understand how card payments work, see [Introduction to Card Payments.](../Key_Concepts/Understanding_Payments.htm)
* If you want to find out more about the Thredd platform, see [The Role of Thredd](../Key_Concepts/Our_Role.htm) and [Thredd Architecture](../Key_Concepts/System_Architecture.htm).
* If you are looking to set up a card program and want to understand what this involves, see [Setting up a Program with Thredd](../Key_Concepts/Setting_up_Program.htm) and [Use Case Scenarios](../Key_Concepts/Scenarios.htm).
