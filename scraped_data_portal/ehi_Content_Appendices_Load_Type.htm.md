# 4.15 Load Type

The table below lists the valid values for the `LoadType` field, identifying the type of Load. (See also [Get Transaction Message fields: LoadType)](../Requirements/GetTransaction_Message.htm#LoadType)

| ID | Source |
| --- | --- |
| 0 | Unknown |
| 1 | Cash |
| 2 | Debit card |
| 3 | Credit card |
| 4 | e-Wallet |
| 5 | Bank account |
| 6 | Import |
| 7 | Savings Stamps |
| 8 | Cheque |
| 9 | Export |
| 10 | Transfer |
| 11 | From/To Offline Balance |
