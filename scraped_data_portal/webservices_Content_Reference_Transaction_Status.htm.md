# Transaction Status

Refer to the table below for a list of transaction status values for the `<StatusCode>` field.

| Type | Description |
| --- | --- |
| A | Accepted |
| C | Cleared |
| I | Declined |
| R | Removed |
| S | Settled |
| V | Reversed |

See usage in: [Card Statement,](../Web_services_api/Card_Statement_v1.htm)  [Card Statement (V2)](../Web_services_api/Card_Statement_v2.htm)
