# Card Regenerate Image

API: `Ws_Regenerate`

This web service enables you to retrieve the card image configured on the Thredd platform, which can then be displayed to the cardholder. Card images are stored for both virtual cards and聽physical cards that have been converted from virtual cards. If a customer wants to see the image some time after card creation, you can regenerate the image by using this web service.
