# About This Document

This guide describes how to set up and manage card fees for your card products on the Thredd system. It includes details of the card usage and administration fees that you can create for your card products, and how to view fees and apply fees to cards.

## Target Audience

This guide is intended for Thredd clients ([Program Managers![Closed](../../Skins/Default/Stylesheets/Images/transparent.gif) A Thredd customer who manages a card program. The program manager can create branded cards, load funds and provide other card or banking services to their end customers.](#)) who are interested in setting up fees for their card products.

## How to use this Guide

If you are new to Thredd and want to understand how card usage fees work and some of the available fee options, see the [Overview](Introduction.htm#_Overview) section.

For information about options for viewing and managing fees using Thredd REST-based Cards API, see [Fee Maintenance (Cards API)](Fee_Maintenance_REST.htm#_Fee_Maintenance).
