# Processing Codes (DE003)

Refer to the table below for a list of processing codes returned in the `<ProcCode>` field for a [Card Statement](../Web_services_api/Card_Statement_v1.htm) request. These codes use the `DE003` messages for Mastercard or Visa transactions. Thredd uses this field to supply a processing code for Thredd-generated transactions; The codes are listed in the **Processing Code** screen on Smart Client. In Thredd Portal, these codes are also listed in a drop-down for filtering transactions.
