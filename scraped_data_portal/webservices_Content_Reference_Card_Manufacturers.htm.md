# Card Manufacturers

Thredd provides an XML file-based interface which allows card manufacturers to accept card generation files from Thredd. For details, see the Thredd [Card generation Interface Specification](https://docs.thredd.ai/Card_Generation_Interface_Specification.htm).

Before submitting a generate card request to Thredd using the [Card Create](../Web_services_api/Card_Create.htm) web services API, make sure you have discussed your card requirements with your selected card manufacturer. You should understand what manufacturer information you need to include in your request.
