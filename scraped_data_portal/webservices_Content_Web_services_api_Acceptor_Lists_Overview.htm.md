# Acceptor Lists Overview

Acceptor lists enable you to control which merchants or merchant code are blocked or allowed to use your service. You can set up multiple acceptor lists and link cards to a list.

You can use the following web services to view and manage your card *Disallow* lists and *Allow* lists:

|  |
| --- |
| [Card Acceptor Disallow list](Card_Acceptor_Blacklist.htm): `Ws_CardAcceptorBlacklist` |
| [Card Acceptor Allow list](Card_Acceptor_Whitelist.htm): `Ws_CardAcceptorWhiteList` |
| [Card Change Acceptor List](Card_Change_Acceptor_List.htm): `Ws_Card_Change_Cardacceptor_List` |
| [Card Change Acceptor List (Bulk)](Card_Change_Acceptor List_1.htm): `Ws_Change_Cardacceptor_List` |
