# Device Types

The `<Device_Type>` tag uses the following device type codes when requesting a payment token.

| Code | Description |
| --- | --- |
| A | Clothing or apparel |
| B | Media or Gaming device, e.g., XBox, TV, set-top box |
| C | Card (no further detail) |
| D | Domestic application, e.g., fridge, washing machine |
| F | Fob or key-fob |
| G | Mobile tag, case or sleeve |
| H | Fashion accessory, e.g., handbag, glasses |
| J | Jewellery, e.g., necklace, rings, bracelets |
| M | Mobile phone (no further detail) |
| P | Personal computer or laptop |
| R | Wristband (no further detail.). Possibly includes rings too. |
| S | Sticker |
| T | Tablet (no further detail) |
| U | Unknown |
| V | Vehicle |
| W | Watch (no further detail) |
| X | Mobile phone or tablet (sender unclear) |
