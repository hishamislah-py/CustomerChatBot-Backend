# Update Cardholder Details (V1)

API: `Ws_Update_Cardholder_Details`

This web service allows you to update cardholder details, such as: `name` and `address`.

You can also use `Ws_Update_Cardholder_Details_V2`, which provides additional fields. See [Update Cardholder Details (V2)](Update_Cardholder_Details_v2.htm).

You must use `Ws_Update_Cardholder_Details_V2` if you want to update the card profile ID (`ProductRef` field) used for card tokenisation.
