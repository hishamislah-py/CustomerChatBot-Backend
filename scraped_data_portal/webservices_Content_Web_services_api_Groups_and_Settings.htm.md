# Thredd Groups and Products

Thredd enables you to set up limit groups, which control your customer card usage. These groups can be set up by Thredd or you may be able to set up and configure them in [Smart Client![Closed](../../Skins/Default/Stylesheets/Images/transparent.gif) Smart Client is Thredd's user interface for managing your account on the Thredd Platform. You install Smart Client as a desktop application which requires a secure connection to Thredd systems in order to access your account.](#) and in [Thredd Portal![Closed](../../Skins/Default/Stylesheets/Images/transparent.gif) Thredd Portal is Thredd's new user interface for managing your cards and transactions on the Thredd Platform.](#). You can use the following web services to view and manage your card groups and products:

|  |
| --- |
| [List Groups](List_Groups.htm): `Ws_list_group` |
| [Card Change Groups (single)](Card_Change_Groups.htm): `Ws_Card_Change_Groups` |
| [Card Change Groups (Bulk)](Card_Change_Groups_1.htm): `Ws_Change_Groups` |
| [List Products](List_Products.htm): `Ws_List_Products` |
