# Cardholder Details Enquiry

API: `Ws_CardHolder_Details_Enquiry` and `Ws_CardHolder_Details_Enquiry_V2`

This web service returns the details of a cardholder.

V2 of this web service has some additional parameters and does not return the details of archived cards. See below.
