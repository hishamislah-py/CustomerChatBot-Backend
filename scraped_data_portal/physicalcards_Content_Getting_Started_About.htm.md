# 1 About this Guide

This guide provides information on how to set up and configure your card programme to support physical (printed) card production.

## 1.1 Target audience

This guide is aimed at developers who need to understand how to set up their physical card products and use the API to create and update the cards in their programme.

## 1.2 What's changed?

If you want to find out what's changed since the previous release, see the [Document History](../Document_History.htm) section.

## 1.3 How to use this Guide

Before you start:

* If you want to understand how card products work on the Thredd system, read the [Overview](Overview.htm).
* For details of how to go about setting up your card products, see [Setting up your Card Products](../Setup/Setup_card_products.htm).
* For information on supported card manufacturers and how to work with them, see [Working with Card Manufacturers](../Setup/Working_with_card_manufacturers.htm).
* For information on how to use the Thredd web services API or the Cards API to create and manage your cards, see [Creating Cards](../Managing/Creating_cards.htm) and [Managing Cards](../Managing/Managing_cards.htm).
