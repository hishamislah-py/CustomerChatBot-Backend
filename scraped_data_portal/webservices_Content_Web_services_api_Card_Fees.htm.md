# Card Fees Overview

You can use the following web services to view and manage your card fees:

|  |
| --- |
| [Card List Pending Fees](Card_List_Pending_Fees.htm): `Ws_List_Pending_Fees` |
| [Card Apply Fees](Card_Apply_Fees.htm): `Ws_Generic_Fees` |
| [Apply FX Rate](Apply_FX_Rate.htm): `Ws_Client_FX` |

For more information on the setting up of fees and restrictions which may apply, see the Thredd [Fees Guide](https://docs.thredd.ai/Fees_Guide.htm).
