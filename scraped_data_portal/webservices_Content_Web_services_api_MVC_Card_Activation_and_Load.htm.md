# MVC Card Activation and Load

API: `Ws_Activate_MVCLoad`

This web service allows simultaneous load and activation of a [Master Virtual Cards![Closed](../../Skins/Default/Stylesheets/Images/transparent.gif) A Thredd virtual card that is restricted to loading and unloading to a physical card and cannot be used for e-commerce or in-store transactions.
An MVC is used to reflect the value of the âactualâ money in the Issuer's bank account. An MVC guarantees that the load is limited to the amount prefunded (i.e. loaded onto MVC) and gives the Programme Manager the ability to distribute funds immediately rather than having to wait for notification of each individual load into the Issuer Bank account.](#) (MVC). It combines the functions of `Ws_BalanceTransfer` and `Ws_Activate` for clients that are using MVCs.
