# 24 Managing User Access

The Access Management menu on the [Apata Portal Dashboard](Apata_dashboard.htm#Apata_Portal_Dashboard) enables you to set up and manage user accounts. Options include:

* [Viewing Users](#Viewing)
* [Viewing User Roles](#Viewing2)
* [Viewing and Creating User Invitations](#Viewing3)

## 24.1 Viewing Users

1. To view a list of users, select Access Management > Users from the left-hand menu.

   [![](../Resources/Images/Apata/users.png "Users screen")](../Resources/Images/Apata/users.png)

   Figure 36: View Users
2. To view details of a user, click the ![](../Resources/Images/Apata/View_details.png) icon.

## 24.2 Viewing User Roles

1. To view a list of available user roles, select Access Management > User Roles from the left-hand menu.

   [![](../Resources/Images/Apata/roles.png "Users roles screen")](../Resources/Images/Apata/roles.png)

   Figure 37: View User Roles
2. To view details of a user role, click the ![](../Resources/Images/Apata/View_details.png) icon.

Your organisation will be provided with a preconfigured list of standard roles (Administrator, Analyst and Customer Service). You will be able to assign these roles to your users.

If you require customised roles, please contact your Thredd 3D Secure Project Manager to discuss. This changes will be chargeable.

## 24.3 Viewing and Creating User Invitations

1. To view a list of users who have been invited to the Apata Portal, select Access Management > Invitations from the left-hand menu.

   [![](../Resources/Images/Apata/invitations.png "User invitation screen")](../Resources/Images/Apata/invitations.png)

   Figure 38: View User Invitations
2. To filter the list of user invitations, select a status button: All, Active, Expired, Revoked and Accepted.
3. To view details of a user invitation, click the View button.
4. To revoke an invitation, click the Revoke button.
5. To create a new user invitation, click the Create User Invitation button.
