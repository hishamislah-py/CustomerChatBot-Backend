# Card Balance Overview

You can use the following web services to check, update or transfer the balance on an account:

|  |
| --- |
| [Card Balance Enquiry](Card_Balance_Enquiry.htm): `Ws_Balance_Enquiry` and `Ws_Balance_Enquiry(_V2)` |
| [Card Balance Enquiry (lightweight)](Card_Balance_Enquiry_1.htm): `Ws_Card_BalEnq` |
| [Card Balance Adjustment:](Card_Balance_Adjustment.htm) `Ws_BalanceAdjustment` |
| [Card Balance Update:](Card_Balance_Update.htm) `Ws_BalanceUpdate` |
| [Card Balance Transfer:](Card_Balance_Transfer.htm) `Ws_BalanceTransfer` |
