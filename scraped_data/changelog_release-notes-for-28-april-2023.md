# Release Notes for 28 April 2023

New changes to the Thredd Cards API for the week ending 28th April 2023.

* [Rebrand of the Cards API documentation](#rebrand-of-the-cards-api-documentation)
* [Program Manager endpoints temporarily removed](#program-manager-endpoints-temporarily-removed)

## Rebrand of the Cards API documentation

We have rebranded our Cards API documentation to reflect our new company name and brand identity.

GPS is now Thredd. [Find out more.](https://www.thredd.com/gps-is-now-thredd)

## Program Manager endpoints temporarily removed

The following endpoints have been temporarily removed from the Cards API:

* Send SMS
* Get Customer Specific Cards
* Get list of cardholders
* Get a list of cards belonging to a given cardholder
