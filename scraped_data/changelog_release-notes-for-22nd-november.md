# Release Notes for 22nd November

* Updated the example on [Introduction to Fraud Alert Service](https://cardsapidocs.thredd.com/docs/introduction-to-fraud-alerts) to remove `CardholderName`.
* In the [Card Bulk Create Endpoint](https://cardsapidocs.thredd.com/docs/bulk-create-cards), the maximum number of cards that can be created in a single request has been increased from 50 to 1000. We have also introduced a 30-minute frequency gap between multiple bulk card creation requests.
