# Cardholder Details - Field Descriptions

The following page details each of the fields that are received in the response when using the Cardholder Details endpoint.

## Response Fields

The following table describes fields that included in the response after successfully retrieving cardholder details.

| Field       | Description                          |
| :---------- | :----------------------------------- |
| title       | The title of the cardholder.         |
| firstName   | The first name of the cardholder.    |
| middleName  | The middle name of the cardholder.   |
| lastName    | The surname of the cardholder.       |
| dateOfBirth | The date of birth of the cardholder. |
| mobile      | The mobile number of the cardholder. |
| email       | The email address of the cardholder. |
