# Accessing the API

The Thredd Cards API provides you with a test environment that you can use to test and develop against the API.

You can access our test environment using the following url:

```json UAT URL
http://cardsapi-uat-pub.globalprocessing.net/
```

> 🚧 Test Environment Credentials
>
> To gain access to the cards API, contact your Thredd Implementation Manager / Thredd Representative who will generate your credentials.
