# Release Notes for 7th February

The following change has been made to the Cards API:

## New Load/Unload Card for Issuer endpoint

A new endpoint, enabling issuers to load and unload funds for cards, is available in this release. This enables users with an Issuer ID linked to their Program Manager to load and unload funds for their cards.

For more information on the Load/Unload Card for Issuer endpoint, see [Load and Unload Card for Issuer](https://cardsapidocs.thredd.com/docs/load-and-unload-bin-sponsor).
