# Release Notes for the 20th February

The following change has been made to Cards API:

### Added Open Banking Endpoints to API Hub

New endpoints have been added to support Thredd's Open Banking solution. Open Banking is a service that enables third parties to obtain financial data from a wide range of financial institutions such as banks, fintechs and credit card companies.

The following endpoints have been added:

* [Register Open Banking Customer](https://cardsapidocs.thredd.com/docs/register-open-banking)
* [Get All Customer Accounts](https://cardsapidocs.thredd.com/docs/get-all-customer-accounts)
* [Get Customer Account Details](https://cardsapidocs.thredd.com/docs/get-customer-account-details)
* [Get Institution Branding](https://cardsapidocs.thredd.com/docs/get-institution-branding)

For more information on Open Banking, see [Introduction to Open Banking](https://cardsapidocs.thredd.com/docs/introduction-to-open-banking).

### Added New Webhook Event Code

A new event code has been added to Thredd's webhook solution. Event Code 117 enables clients to be notified when  a customer interacts with an Open Banking registration session.

For more information, see [Webhook Event Codes](https://cardsapidocs.thredd.com/v2.0/docs/webhook-event-codes-1#event-code-117---open-banking-registration).
