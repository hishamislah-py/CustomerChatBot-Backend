# Release Notes for 12th January 2024

New change to the Thredd Cards API for the week ending 12th January 2024.

## Sandbox updated

The Sandbox has been updated to include all changes available in the UAT as of the 15th December 2023 release. The Sandbox environment now includes all changes included in the release for:

* [3rd November 2023 (Excluding the 3DS endpoint - 3DS is not supported in the Sandbox environment)](https://cardsapidocs.thredd.com/v0.0/changelog/release-notes-for-3rd-november-2023)
* [15th December 2023 (Excluding changes made to the 3DS endpoint)](https://cardsapidocs.thredd.com/v0.0/changelog/release-notes-for-15th-december-2023)
* [PRN-165](https://cardsapidocs.thredd.com/v0.0/changelog/prn-165-1)

For more information on using the Sandbox environment, see [Gaining Access to the Developer Portal.](https://cardsapidocs.thredd.com/v0.0/docs/gaining-access-to-the-developer-portal)
