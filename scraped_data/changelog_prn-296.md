# PRN-296

The Get Card Data endpoint has been updated to use new RSA-4096 keys with OAEP/PSS-SHA-256 padding for enhanced security.

> 📘 More Information
>
> For more information on the Get Card Data endpoint and how to encrypt keys with the new key pair, see [Introduction to Sending Secure Data](https://cardsapidocs.thredd.com/docs/introduction-to-sending-secure-data) and [Get Card Data](https://cardsapidocs.thredd.com/docs/get-encrypted-data).
