# Release Notes for the 23rd January 2026

The following change has been made to Cards API:

## New Card Limits Endpoints Added

Two new endpoints have been added to Core API and API Hub.

* The Get Card Limits endpoint enables you to retrieve the velocity limits associated with a specific card.
* The Set Card limits endpoint enables you to update the velocity limits of a Thredd card.

> 📘 More information
>
> * For more information on these new endpoints, see [Introduction to Card Limits](https://cardsapidocs.thredd.com/docs/introduction-to-card-limits).
> * To view these endpoints in the API Explorer, see [Set Limit](https://cardsapidocs.thredd.com/reference/set_limit_set_limit_post) and [Get Limit](https://cardsapidocs.thredd.com/reference/get_limit_get_limit_get).
