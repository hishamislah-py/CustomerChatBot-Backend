# Release Notes for the 2nd April

The following change has been made to Cards API:

### Updated maximum length of the firstName field in the Create Card endpoint

The `firstName` field in the Create Card endpoint has been extended from 20 characters to 30 characters. This change is applicable to Core API and Cards API.

### Obsolete Action Codes removed from the SOAP Converted Endpoints response

All obsolete action codes related to the SOAP Converted Endpoints have been removed. See [Action Codes](https://cardsapidocs.thredd.com/docs/action-codes) for a full list of action codes available.
