# Release Notes for the 17th April

The following change has been made to Cards API:

### New Webhook Event Code

Event Code 111 has been added to our webhook functionality for this release, supporting notifications for when Apata sends a finalised event.

For more information on webhooks, see [Introduction to Webhooks](https://cardsapidocs.thredd.com/docs/introduction-to-webhooks).

### Added Support for Web Provisioning

This release sees support for Web Push Provisioning with the new Apple Web Push Provisioning endpoint, and the updated Google Unified Push Provisioning endpoint. For more information, see [Apple Web Push Provisioning](https://cardsapidocs.thredd.com/docs/apple-web-push-provisioning) and [Google Unified Push Provisioning](https://cardsapidocs.thredd.com/docs/request-google-payload).
