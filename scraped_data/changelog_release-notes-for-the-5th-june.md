# Release Notes for the 5th June

The following change has been made to Cards API:

### New Create Microdeposit Verification Link Endpoint Added

The Create Microdeposit Verification Link endpoint has been added to API Hub. The endpoint enables you to generate a Mastercard AVA Data Connect URL used by end users to verify microdeposit amounts.

For more information on the endpoint, see [Create Microdeposit Verification Link](https://cardsapidocs.thredd.com/docs/create-micro-deposit-verification-link).
