# Transaction XML Reports

Thredd provide a number of daily XML reports, containing details of transactions processed on the system during the past 24 hours. This includes both authorisations and financial transactions.

Reports are provided by Secure File Transfer Protocol (sFTP).

<Image alt="XML Reporting" align="center" border={true} src="https://files.readme.io/8df9622-XML_reporting.png">
  XML Reporting
</Image>

The Transaction XML report provides details of authorisation and financial advices and any other non-scheme transactions that take place, such as card loads/unloads, balance adjustments (via Thredd API), cardholder fees generated by Thredd (that you have set up in your fee groups), and network scheme fees.\
You can use this report to reconcile against Mastercard or Visa settlement advices, and to reconcile with data received from EHI.
