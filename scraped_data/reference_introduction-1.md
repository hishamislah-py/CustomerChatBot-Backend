# API Explorer

## Introduction

Welcome to the API Explorer.

Use this section to explore the API and download a sample Postman Collection to help get you started quickly.

> 📘 Note
>
> The endpoints in the API Explorer and the Postman Collection use the public UAT base URL. For example:
>
> ```json Example UAT Base URL
> https://cardsapi-uat-pub.globalprocessing.net/api/v1/
> ```

This section also explains how to authenticate to the API using OAuth2, and about the headers required in each request. API versioning, error handling and our fair use policy are also described.
