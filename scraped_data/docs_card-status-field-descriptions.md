# Retrieve Card Status - Field Descriptions

The following page details each of the fields that can be returned in the response when using the Retrieve Card Status endpoint.

## Response Fields

The following table describes fields that included in the response after successfully retrieving a card status

| Field                 | Description                                    |
| :-------------------- | :--------------------------------------------- |
| status                | The current status of the card.                |
| cardStatusCode        | The current two-digit status code of the card. |
| cardStatusDescription | The current code and status of the card.       |
