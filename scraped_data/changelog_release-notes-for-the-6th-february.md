# Release Notes for the 6th February

The following change has been made to Cards API:

### New Fraud Login Event Endpoint Added for API Hub

The Fraud Login Event endpoint has been added to API, enabling customers that use Fraud Transaction Monitoring to send a notification whenever a cardholder logs in. This enables customers to detect more fraud (especially the account takeover fraud type) and reduce false positive rates by introducing more granular rules.

For more information on the endpoint, see [Fraud Login Event](https://cardsapidocs.thredd.com/docs/fraud-login-notification).
